# Samsara Telegram Bot + Gemini

Бот для Telegram, который по номеру трака показывает активные ошибки из Samsara
и расшифровывает коды через Google Gemini.

## ENV-переменные

На Vercel должны быть заданы:

- `TELEGRAM_BOT_TOKEN` — токен бота от @BotFather.
- `SAMSARA_API_KEY` — API ключ Samsara (Bearer-токен).
- `GEMINI_API_KEY` — API ключ Google Gemini.

## Что умеет ИИ-часть

1. Собирает коды и описания ошибок из Samsara.
2. Отправляет их в Gemini с промптом:
   - Пояснить по-русски, что за ошибка,
   - типичные причины,
   - насколько срочно чинить,
   - можно ли ехать дальше или лучше остановиться.
3. Возвращает краткое объяснение, которое бот добавляет под списком ошибок.
